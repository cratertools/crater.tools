# Crater Tools

Official website for Crater Tools iOS applications.

## Code Formatting

### HTML Blank Line Standards

All HTML files in this project follow these blank line formatting rules:

**1. No blank lines after opening tags**
```html
<div class="row">
    <div class="col">
```

**2. No blank lines before closing tags**
```html
        <p>Content here</p>
    </div>
</div>
```

**3. Blank lines between sibling containers**
```html
<div class="col">
    <div class="card">...</div>
</div>

<div class="col">
    <div class="card">...</div>
</div>
```

**4. Blank lines between major sections**
```html
<div class="col-12 mb-5">
    <!-- Features section -->
</div>

<div class="col-12 mb-5">
    <!-- Screenshots section -->
</div>
```

**5. Definition lists: blank lines between pairs, not within**
```html
<dl>
    <dt>Platform</dt>
    <dd>iOS</dd>

    <dt>Price</dt>
    <dd>Free</dd>
</dl>
```
